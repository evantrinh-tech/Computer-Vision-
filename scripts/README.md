# Scripts Directory

Thư mục này chứa các script tiện ích để chạy hệ thống.

## Batch Scripts (.bat)

- `he_thong.bat` - Menu chính để chạy hệ thống
- `tao_venv.bat` - Tạo virtual environment
- `setup_tensorflow.ps1` - Cài đặt TensorFlow (PowerShell)

## Python Scripts

Các script Python chạy trực tiếp được đặt ở thư mục gốc để dễ truy cập.
